# Changelog

All notable changes to the ApexFlow platform will be documented in this file.

## [1.0.0] - 2026-01-03

### Added
- **Core Architecture**: Modular Python package structure.
- **ML Engine**: XGBoost training with Optuna hyperparameter optimization.
- **Data Versioning**: DVC integration for telemetry CSVs.
- **Auto-Retraining**: Prefect flows triggered by drift alerts.
- **API Service**: FastAPI prediction service with API key auth and rate limiting.
- **Containerization**: Multi-stage Docker build for production deployments.
- **Observability**: Prometheus/Grafana/Loki stack for monitoring.
- **Documentation**: Comprehensive knowledge base, guides, and diagrams.

## [0.1.0] - 2025-12-27
- Initial project structure and `.gitignore` creation.
